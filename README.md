# Artesanía Mundial Marketplace

Aplicación Node.js con Express, EJS y MySQL desplegada en Render.

## Variables de entorno necesarias

- `DB_HOST`
- `DB_USER`
- `DB_PASS`
- `DB_NAME`
- `DB_PORT`
- `BASE_URL`
- `EMAIL_USER`
- `EMAIL_PASS`
- `EMAIL_FROM`
- `EMAIL_TO`
- `SEND_API_KEY`
- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
- `TWILIO_WHATSAPP_FROM`
- `TWILIO_WHATSAPP_TO`


## Comando de inicio

```bash
npm install
npm start
