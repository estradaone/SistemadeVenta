# Artesanía Mundial Marketplace

Aplicación Node.js con Express, EJS y MySQL desplegada en Render.

## Variables de entorno necesarias

- `DB_HOST`
- `DB_USER`
- `DB_PASS`
- `DB_NAME`
- `DB_PORT`
- `BASE_URL`
- `EMAIL_PASS`
- `EMAIL_FROM`
- `EMAIL_TO`
- `SEND_API_KEY`

## Comando de inicio

```bash
npm install
npm start
